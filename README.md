# ♟️ Chess Player Performance Analysis

## 📖 Overview

This project analyzes the performance of the world's **Top 20 chess players** from **January 2000 to January 2021** using **Python** and **Looker Studio**. The objective is to evaluate player performance based on ELO ratings, age, ranking position, and appearances while building an interactive dashboard to uncover meaningful insights.

---

## 🎯 Project Objectives

* Analyze the performance of top-ranked chess players.
* Study the relationship between Age and ELO ratings.
* Identify players with the highest ELO ratings.
* Find players with the most appearances in the Top 20 rankings.
* Compare player performance across different age groups.
* Build an interactive dashboard for performance analysis.

---

## 💼 Business Problem

Chess organizations, sports analysts, and researchers need to evaluate player performance over time. By analyzing ELO ratings, player age, ranking position, and appearances, they can identify consistent performers, emerging talents, and trends that influence competitive success. This analysis supports strategic decision-making and performance evaluation.

## 💡 Key Insights

* Magnus Carlsen achieved the highest ELO rating in the dataset.
* The dataset includes players aged **17 to 51 years**.
* Several legendary players consistently appeared in the Top 20 rankings over multiple years.
* Age alone does not determine a player's ELO rating.
* Bubble chart analysis highlights the relationship between Age, ELO, and player appearances.
* Average ELO remains competitive across different age groups.
