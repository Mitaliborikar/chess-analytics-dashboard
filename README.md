♟️ Chess Players Performance Analysis
📖 Overview

This project analyzes the performance of the Top 20 world chess players from January 2000 to January 2021 using Python and Looker Studio. The objective is to explore player performance, ELO ratings, age distribution, and player consistency over multiple years through an interactive dashboard.

🎯 Project Objectives
Analyze the performance of top-ranked chess players over 22 years.
Study the relationship between player age and ELO rating.
Identify the highest-rated players across different years.
Analyze player consistency based on repeated appearances.
Build an interactive dashboard to visualize player performance trends.
💼 Business Problem

Chess organizations, analysts, and sports researchers need to evaluate player performance over time to identify consistent top performers, understand rating trends, and study how factors such as age influence ELO ratings. By analyzing historical rankings, stakeholders can compare player performance, identify long-term consistency, and gain valuable insights into competitive chess.
